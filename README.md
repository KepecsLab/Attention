This GitHub repository contains the code for behavioral control, in vivo electrophysiological recording, fiber photometry recording, optogenetic manipulation, and data analysis for the BF-PV sustained attention project.

## Repository structure
- `ephys_acquisition_optostim/`: behavioral task control, ephys recording, opto-tagging scripts
- `ephys_analysis/`: ephys data preprocessing, and figure-generation scripts
- `photometry_acquisition_optostim/`: behavioral task control, fiber photometry recording, optogenetic manipulation scripts
- `photometry_analysis/`: preprocessing, and figure-generation scripts
- `ephys_data/`: small example ephys datasets
- `photometry_data/`: small example photometry datasets
- `external_code/`: code adapted from other public repositories

## Data availability
Full recorded datasets are not stored in this GitHub repository because of file size. They are available on the DANDI Archive at the following URL: https://dandiarchive.org/dandiset/001867.

## External code
Some code may be adapted from other public repositories. Original source links, authors, licenses, and modifications are documented in the relevant folders.
